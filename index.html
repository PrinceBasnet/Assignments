Full Stack Development

Full stack development refers to the development of both front-end (client side) and back-end (server side) portions of a web application. A full stack developer has the skills to work with databases, servers, APIs, and client-side technologies.

  • Front-end: HTML, CSS, JavaScript, Bootstrap
  • Back-end: Node.js, Express, Python, Java, etc.
  • Database: MySQL, MongoDB, PostgreSQL, etc.

HTML

HTML (HyperText Markup Language) is the standard markup language for creating web pages. It provides the structure for web content using elements like headings, paragraphs, lists, tables, images, and forms.

  • <html> → Wraps the whole webpage content.
  • <head> → Contains information about the page (title, links, metadata).
  • <body> → Contains everything users see on the webpage.
  • <h1> â€¦ <h6> → Headings (H1 is the biggest, H6 is the smallest).
  • <p> → Paragraph of text.
  • <a href="URL"> → Creates a link to another page or site.
  • <img src="image.jpg" alt="description"> → Displays an image.
  • <ul> and <ol> → Unordered (bullets) and ordered (numbers) lists.
  • <table>, <tr>, <td>, <th> → Used to create tables with rows and cells.
  • <form> → Collects user input (e.g., text fields, checkboxes, buttons).

CSS

CSS (Cascading Style Sheets) is used to style and layout web pages. It controls the color, font, spacing, and positioning of HTML elements, making websites visually appealing and responsive.

  • color → Sets text color.
  • background-color → Changes background color.
  • font-family → Changes the style of the text (e.g., Arial, Verdana).
  • margin → Space outside an element.
  • padding → Space inside an element (around content).
  • border → Adds a border around an element.
  • width / height → Sets size of elements.
  • display (block, inline, flex, grid) → Controls layout behavior.
  • text-align → Aligns text (left, right, center).
  • :hover (pseudo-class) → Changes style when mouse hovers over an element.

Bootstrap

Bootstrap is a popular front-end framework for building responsive and mobile-first websites quickly. It provides ready-to-use components like navigation bars, buttons, forms, and grids.

  • .container → Wraps content with proper spacing.
  • .row → Creates a horizontal group of columns.
  • .col- (e.g., .col-md-6) → Defines how wide a column is (responsive).
  • .btn, .btn-primary, .btn-success → Pre-styled buttons.
  • .card → A content box with padding and shadow.
  • .navbar → Creates a responsive navigation bar.
  • .table, .table-striped, .table-bordered → Pre-styled tables.
  • .text-center, .text-primary, .bg-light → Text and background utilities.
  • .alert, .alert-warning, .alert-success → Alert boxes for messages.
  • .form-control → Styles input fields and forms.

JavaScript

JavaScript is a versatile programming language that enables interactive and dynamic features on web pages. It can manipulate HTML and CSS, handle events, validate forms, and communicate with servers.

  • document.getElementById("id") → Selects an element by its ID.
  • document.querySelector("selector") → Selects an element using CSS-style selectors.
  • innerHTML → Changes the content inside an element.
  • style.property → Changes CSS with JS (e.g., element.style.color="red").
  • addEventListener("click", function) → Runs code when a user clicks.
  • alert("message") → Shows a popup message.
  • console.log("text") → Prints to the browser console (for debugging).
  • if...else → Runs different code depending on conditions.
  • for / while → Loops that repeat code.
  • function myFunction() → Groups reusable code.
